Authentication

The LoanEngine API uses API keys to authenticate requests. You can view and manage your API keys in the Dashboard. Pass the key in the header of each request.

Authorization: Bearer pk_live_...
warning

Your API keys carry many privileges, so be sure to keep them secure! Do not share your secret API keys in publicly accessible areas such as GitHub, client-side code, and so forth.

POST

/v1/score/calculate

Calculate the credit score for a given applicant. This endpoint accepts a payload containing personal information, income data, and loan request details. It returns a score between 300 and 850 along with risk factors.

Body Parameters

Parameter Type Description
applicant_id REQUIRED string Unique identifier for the applicant in your system.
annual_income REQUIRED integer Total annual income in USD cents.
employment_status enum One of: full_time, part_time, self_employed.
loan_amount integer Requested loan amount in USD cents. Default is 0.
terminal Request
curl -X POST https://api.loanengine.io/v1/score/calculate \ -H "Authorization: Bearer pk_live_8f9s8d..." \ -H "Content-Type: application/json" \ -d '{ "applicant_id": "app_923842", "annual_income": 8500000, "employment_status": "full_time", "loan_amount": 2500000 }'
code Python Client
import requests url = "https://api.loanengine.io/v1/score/calculate" headers = { "Authorization": "Bearer pk_live_...", "Content-Type": "application/json" } data = { "applicant_id": "app_923842", "annual_income": 8500000, "employment_status": "full_time" } response = requests.post(url, headers=headers, json=data) print(response.json())
